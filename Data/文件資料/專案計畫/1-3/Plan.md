# 技術棧定義（Tech Stack）
個人記帳＋月報表網站

---

## 1. 前端（Frontend）

- Framework: Next.js 14（App Router）
- Language: TypeScript
- UI / Styling: Tailwind CSS + shadcn/ui
- State / Data Fetching: TanStack Query（React Query）
- Form / Validation: React Hook Form + Zod
- Routing: Next.js App Router（內建）
- Charts: Recharts
- Date Handling: date-fns

## 2. 後端（Backend）

- Runtime / Framework: Next.js API Routes（Route Handlers）
- Language: TypeScript
- API Style: REST
- Auth: NextAuth.js（Credentials Provider）+ JWT
- Validation: Zod

## 3. 資料庫（Database）

- DB Engine: SQLite（本機單檔）
- ORM: Prisma
- Migrations: Prisma Migrate

## 4. 其他（Optional）

- Testing: Vitest + React Testing Library
- Dev Tooling: ESLint + Prettier + Husky
